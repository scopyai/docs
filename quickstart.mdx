The fastest way to use Scopy AI is to connect GitHub, choose repositories, and open or update a pull request. Scopy reviews enabled repositories automatically and publishes confirmed findings back to GitHub.

## 1. Sign in

Open [app.scopy.dev](https://app.scopy.dev) and sign in.

## 2. Connect GitHub

Choose **Connect your GitHub repositories** and install the Scopy GitHub App on the user account or organization you want to review.

During GitHub installation, select the repositories Scopy should be allowed to see. You can grant access to all repositories or choose specific repositories.

## 3. Choose repositories in Scopy

After returning from GitHub, Scopy shows the repositories available to the GitHub App installation. Enable the repositories where reviews should run.

Scopy only reviews repositories that are enabled in the workspace. If a repository is missing, update the [GitHub App installation](https://github.com/apps/scopyai) to refresh the list of repositories available for review.

## 4. Open a pull request

Create a new pull request or mark a draft pull request ready for review in an enabled repository. Scopy queues a review, analyzes the changed code and publishes confirmed findings back to the pull request page.

You can also request another review by mentioning the Scopy GitHub App in a pull request comment.

## 5. Review results

Scopy results appear in two places:

- GitHub pull request review comments and a summary comment.
- The Scopy dashboard, where you can inspect pull requests, review state, findings and workspace analytics.
